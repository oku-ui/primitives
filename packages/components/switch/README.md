# `@oku-ui/switch`

## Installation

```sh
$ pnpm add @oku-ui/switch
```

## Usage

```ts
<script>
import { OkuSwitch, OkuSwitchThumb } from '@oku-ui/switch';
<script>

<template>
    <OkuSwitch id="switch" v-model="checked" class="switchStyle">
      <OkuSwitchThumb class="thumbStyle" />
    </OkuSwitch>
</template>
```
